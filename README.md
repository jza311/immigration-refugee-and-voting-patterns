# Relationship Between Immigration and Partisanship in Canada

## Project Overview

This project examines how immigration attitudes influence political partisanship and whether large refugee inflows affect voting intentions in Canada.

Using the 2015 Syrian Refugee Crisis as a quasi-experimental shock, this study investigates whether refugee resettlement increased support for right-wing political parties, particularly Conservative voting intention.

The analysis combines immigration sentiment analysis with a Difference-in-Differences (DiD) design to identify both correlation and causal effects.

---

## Methods Used

- Ordinal Logistic Regression
- Event Study Analysis
- Difference-in-Differences (DiD)
- Survey Weighting
- Political Sentiment Analysis

---

## Tools Used

- R
- Statistical Modeling
- Data Cleaning
- Regression Analysis
- Data Visualization

---

## Key Findings

- Conservative voters had significantly higher odds of agreeing that Canada admitted too many immigrants
- NDP voters showed lower odds compared to Liberal voters
- Surrey experienced an estimated 6.7 percentage point decline in Conservative vote intention after refugee resettlement
- No statistically significant right-wing backlash was found following large-scale Syrian refugee arrivals

## Data Access

Data for this project was sourced from:
- Innovative Research Group (CTM surveys)
- Statistics Canada

Raw survey data is not publicly included in this repository due to access restrictions and data-sharing limitations.

This repository includes methodology, analysis scripts, and summarized outputs only.
